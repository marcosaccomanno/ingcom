Diagramas de flujo y calculos de complejidad:

```mermaid
graph TD
    A[main] -->
    B[contador = 1] -->
    C{while contador <= 5} -->
    |si| D{if contador % 2 != 0} -->
    |si| E(print contador) -->
    F
    D -->
    |no| F[contador += 1] -->
    C
    C -->
    |no| G[end]
```
* *main*:
    * V(G) = R = 3
    * V(G) = A-N+2 = 8-7+2 = 3
    * V(G) = P+1 = 2+1 = 3
